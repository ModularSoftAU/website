## Introduction
This feature allows linked players to edit their player profiles to add things from their Social Media handles, interests and more.

### Profile Options
Currently we support the following profile options

* About
* Interests
* Social Media
    * Discord
    * Steam
    * Twitch
    * YouTube
    * Twitter
    * X/Twitter
    * Spotify
    * Reddit
